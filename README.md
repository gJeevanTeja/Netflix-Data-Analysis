# Netflix-Data-Analysis
I recently completed a data visualization project using the Netflix dataset to uncover how the streaming giant’s library has evolved over the years.

I followed the industry level path 
1. EDA in python 
2. load to SQL database 
3. connect with Power bi 
4. Documentation of my work
5. Presentation 

Key Insights i have Identified: 
📍 Content Peak: 2018 was a massive year for Netflix, with the highest volume of content added. 
📍 Top Producer: The United States remains the leading contributor to the global library.
📍 Viewer Preference: Documentaries emerged as a top-performing genre. 📍 Content Rating: A significant portion of the library (over 2,800 titles) is rated TV-MA, highlighting a focus on adult audiences.
